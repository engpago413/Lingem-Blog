# 关于我

Lingem，可以叫我凌哥。这个名字其实没有什么深意，就是取了「灵阁」的谐音，泛指神圣的宫殿、神妙的楼阁。

我在现实生活中没有那么多的朋友，也不是一个擅长社交的人，更喜欢一个人独来独往。别人可能觉得独处很无聊，但我反而很享受这种安静的状态。戴上耳机，打开电脑，沉浸在自己的世界里，对我而言就是一种放松。

聊聊我的兴趣爱好吧，我大概从 2012 年左右开始接触到了日本动画，虽然不太记得清我看的第一部作品是什么了，但我当时看的这些作品，<spoiler>缘之空、日在校园、七人魔法使、恶魔高校D×D，</spoiler>确实给我幼小的心灵留下了巨大的震撼。慢慢的，我对日漫越来越着迷，直到现在。目前我主要在看校园恋爱和异世界题材的作品，恋爱作品我最喜欢的是我心危，异世界题材我最喜欢的是转生史莱姆。

除了二次元，能说的上爱好的应该就是折腾一些开源项目了吧，其实就是遇到自己感兴趣的开源项目就拉下来在自己机器上跑跑，然后尝试融入到自己的日常生活中，比如 Emby、Qbittorrent、Immich、openlist 等等。我也说不上为什么喜欢，就是当自己花费不少精力，终于把项目跑通运行起来后，感觉心里很满足。

# 关于这个博客

本博客基于 [Astro](https://astro.build/) 框架与 [Fuwari](https://github.com/saicaca/fuwari) 主题构建，并根据个人喜好做了不少修改，具体的修改内容可以来我的仓库查看。

::github{repo="engpago413/Lingem-Blog"}

这个博客算是我一个人的小小空间。在这里，我可以静静地写一些我感兴趣的文章，分享一些我喜爱的事物。我不需要过分在意他人的眼光，写自己想写的，记录自己想记录的，这就足够了。

博客主要记录自己的折腾过程和一些兴趣爱好。我并不追求成为什么技术博主，只是把自己踩过的坑、学到的东西写下来，一方面方便自己以后回顾，另一方面也许能帮到和我一样在摸索的人。

# 内容方向

目前博客的内容主要围绕以下几个方面：

- **HomeLab**：围绕我的<spoiler>绿联 DX4600-Pro</spoiler> NAS 展开，记录 Docker 部署各类服务的折腾过程，比如 Karakeep 书签管理、Emby 媒体服务器等。
- **二次元**：偶尔分享一些喜欢的动漫、游戏或相关的感想。这一块比较随性，想到什么就写什么。
- **日语学习笔记**：正在自学日语，把学习过程中整理的语法记录在这里，当作自己的学习存档。虽然还在起步阶段，嘛，一步一步来就好。

当然，未来也许还会加入更多感兴趣的话题，谁知道呢。
